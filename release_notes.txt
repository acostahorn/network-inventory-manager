AAA Networking Website
Project: Making a e-shop website for a networking company that provides electronic components sale and installation of infrastructure for small to medium businesses).

Installation instructions.
Before starting the server by typing:
nodemon index.js

Made sure the necessary modules to run the app are installed in the directory, by typing the command
npm i xyz (being xyz the name of the module)

Here is the list of the dependencies:

node.js
express
path
ejs
ejs-mate
nodemon
method-override
mongoose
joi
passport
passport-local
passport-local-mongoose

The app loads and saves data to Mongo.db in the AAAStore database.
Both the products and the users collections are stored on Mongodb.

to start the app enter the following command:
nodemon index.js

and open the browser at the page
https://localhost:8080

The backend code can be run from terminal in a Linux distribution (server). 
It works with Fedora 43 and Debian 12. Debian 13 not fully compatible because there is no obvious way to install mongo shell.
